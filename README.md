# Manchester United: Auditoría de Gasto Neto

🚀 **[Ver Panel Interactivo en Vivo](https://altatbxqv5yf4mcxtqre39.streamlit.app)**

Análisis financiero y métrico del desgaste deportivo en el Manchester United durante la era Post-Ferguson. Este laboratorio utiliza las dinámicas del club para plantear la pregunta fundamental: **¿Cuánto dinero le ha costado al directorio cada punto obtenido por los distintos entrenadores en la última década?**

---

## Métricas Clave y Despliegue

La herramienta funciona como un panel de inteligencia de negocios para toma de decisiones en gerencia deportiva:

- **Gasto Neto por Punto:** Indicador central que divide la inyección financiera neta al mercado de fichajes sobre la retribución de puntos finales en la clasificación. Mide la verdadera eficiencia operativa del líder táctico.
- **Factores de Riesgo (Goles en Contra):** Auditoría correlacional estadística para probar si encajar goles es tan dañino financieramente a largo plazo como la incapacidad de generarlos.
- **Estabilidad de Código:** Entorno robusto mediante refactorización limpia de excepciones vacías o redundantes, garantizando despliegues en la nube en tiempo real sin vulnerabilidades ocultas.

---

## Ejecución Local

Descarga, instala el entorno ligero base y dispara la interfaz gráfica:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

*Los resultados muestran numéricamente la trampa financiera de la gestión corta: cuando las indemnizaciones por técnicos despedidos superan los 32 millones de libras, el club gasta más en re-empezar la estructura que en consolidarla.*
